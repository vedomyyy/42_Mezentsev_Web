## 🇬🇧 English

### About

This repository contains 7 web projects. Projects are organized in folders `1` through `7` and demonstrate progressive skill growth — from static HTML pages to an interactive SPA built with React and TypeScript.

---

### 📁 Project Structure

| Folder | Name | Description | Stack |
|--------|------|-------------|-------|
| `1` | **DnevnikM v1** | Personal diary page: jokes, a video review, and FC Rostov match schedule | HTML, CSS |
| `2` | **ShitTalker67 v1** | Humorous landing page for a "talking toilet" with animations and fake ads | HTML, CSS, JS |
| `3` | **DnevnikM v2** | Revised version of the diary with updated styling | HTML, CSS, JS |
| `4` | **ShitTalker67 v2** | Extended ShitTalker67: review form, dark/light theme toggle, cookie handling | HTML, CSS, JS |
| `5` | **Bestiary-Megabonk** | Creature cards dynamically rendered via JS, with a stats sidebar | HTML, CSS, JS |
| `6` | **Puls.rf** | Community forum: CRUD threads, comments, user profile, filtering, REST API | HTML, CSS, JS |
| `7` | **Puls.rf (React)** | Project #6 rewritten in React + TypeScript | React, TypeScript |
